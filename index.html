The open standard for simple, structured logistics data

Carqo is an open-source logistics data standard that brings clarity and simplicity to the global transport ecosystem. It defines a unified structure for shipments, parties, planning, and reporting, enabling carriers, shippers, platforms, and software vendors to work with a single interoperable logistics schema.

Access the Carqo core on GitHub.
